# Cuestionarios - Del Rio Solutions

Este módulo se agregará a un sistema que ya existe, el módulo cuenta con los siguientes componentes.

  - API (PHP Backend)
  - FrontEnd (Angularjs, HTML5, CSS3)
  - Android App (Angularjs, HTML5, CSS3, Ionic)
  - DB Design
 
### Installation

Para clonar el repositorio es necesario seguir estas instrucciones:
```sh
$ git clone git@github.com:jerrygarciab/del-rio-cuestionario.git
$ cd del-rio-cuestionario
```
